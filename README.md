## üöÄ Getting Started

### Local Development

1. Clone or download this repository
2. Open `index.html` in your web browser
3. No build process needed - it just works!

### Deploy on GitHub Pages

#### Option 1: Create a new repository (Recommended for portfolio)

1. Create a new repository named `portfolio` or `ITSsafer-development` in your GitHub account
2. Push this project to that repository
3. Go to repository **Settings** ‚Üí **Pages**
4. Set source to `main` branch
5. Your site will be live at `https://yourusername.github.io/portfolio`

#### Option 2: Use your GitHub profile page

1. Create a repository named `ITSsafer-DevOps.github.io`
2. Push this project to that repository
3. Your site will be live at `https://ITSsafer-DevOps.github.io`

## üéØ Customization Guide

### Modify Colors

Edit the CSS variables in `styles.css`:

```css
:root {
    --primary-color: #FF3131;      /* Red/brand color */
    --text-primary: #1f2937;       /* Main text color */
    --text-secondary: #6b7280;     /* Secondary text */
    /* ... more variables ... */
}
```

### Update Project Information

Edit the project cards in `index.html` and modify:
- Project title and description
- Technology stack tags
- Features list
- GitHub links

### Modify Navigation Links

Update the navbar links in `index.html`:

```html
<ul class="nav-links">
    <li><a href="#projects">Projects</a></li>
    <li><a href="#about">About</a></li>
    <li><a href="#contact">Contact</a></li>
</ul>
```

### Change GitHub Organization

Replace all instances of `https://github.com/ITSsafer-DevOps` with your GitHub organization URL.

## üìù Deployment Notes

### For GitHub Pages:

1. **Repository Settings**
   - Go to Settings ‚Üí Pages
   - Choose "main" or "master" as source

   - Update DNS settings with your provider
